## Core Value

- **App Router**: File-system routing with layouts, loading states, error boundaries
- **Server Components**: Render on server, ship zero JS to client
- **Server Actions**: Mutate data without API routes
- **Middleware**: Run code before requests complete
- **ISR**: Incremental Static Regeneration
- **Image/Font optimization**: Automatic performance optimization
- **Turbopack**: Next-gen bundler (Rust-based)

## Risks

- Vercel vendor lock-in concerns (optimized for Vercel deployment)
- Complexity increasing with each major version
- Server Components learning curve
- Some community friction around App Router adoption
