## Core Value

- **Compile-time magic**: No virtual DOM, no runtime overhead
- **Reactive by default**: `$state`, `$derived`, `$effect` (Svelte 5)
- **Less code**: Significantly less boilerplate than React
- **SvelteKit**: Full-stack framework (like Next.js for Svelte)
- **Highest DX satisfaction**: Developers who try Svelte love it

## Risks

- Different paradigm from React — parallel skill investment
- Smaller job market and ecosystem
- Fewer ready-made components compared to React
